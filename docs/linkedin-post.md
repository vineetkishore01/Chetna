## 🧠 I built a memory system for AI agents that thinks like a human brain

What happens when your AI forgets everything after every conversation?

I've been working on this problem for months, and I think I finally cracked it.

**Introducing Chetna** (Hindi for "Consciousness")

A memory system for AI agents that doesn't just store data—it thinks like us.

Here's what makes it special:

## The Problem
Most AI memory systems just store → retrieve. Like a filing cabinet.

But human memory doesn't work that way.

When you ask "What does the user like?" — your brain doesn't just search. It considers:
- How important is this? ⚡
- How recent? 🕐
- How often do I recall it? 🔄
- Was it emotional? ❤️

## The Solution
Chetna combines 5 factors like human brains:

```
Recall Score =
  Similarity (40%) +    // How relevant to the query
  Importance (25%) +    // Critical memories stick
  Recency (15%) +      // Recent = easier to recall
  Access Frequency (10%)+ // Frequently accessed = important
  Emotional (10%)       // Emotional memories stick
```

## Real Example

User tells AI: "My name is Wolverine and my human is Vineet"

Later, user asks: "Who owns me?"

Traditional search: ❌ No match (different words)
Chetna semantic search: ✅ "My human is Vineet"

Because it UNDERSTANDS meaning, not just keywords.

## Built for AI Agents

- 📊 **Auto-decay** - Like human forgetting curve
- 🔄 **Consolidation** - LLM reviews and updates importance
