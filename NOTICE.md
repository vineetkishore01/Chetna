## Research Foundations

### Ebbinghaus Forgetting Curve

The memory decay implementation is based on Hermann Ebbinghaus's research on
memory and forgetting.

- **Researcher**: Hermann Ebbinghaus
- **Publication**: "Über das Gedächtnis" (1885)
- **Concept**: Exponential decay of memory retention over time
