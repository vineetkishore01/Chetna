## Contributors

### How to Add Your Name

If you've contributed to Chetna, you're automatically included in this list. To add your name:

1. Make a contribution (PR, issue report, documentation, etc.)
2. Include this line in your PR description: `Please add me to CONTRIBUTORS.md`
3. The maintainer will add you during merge

Alternatively, open an issue requesting to be added.
